# **App Name**: ClassMate

## Core Features:

- Teacher Authentication: Secure portal for teachers to log in using their full name as username and 'aua9ncrc' as password.
- Student Authentication: Secure portal for students to log in using roll number as identification.
- Attendance Input: Teachers manually input attendance for each student (roll numbers 463 to 536) in 7 different classes based on a daily schedule.
- Attendance Tracking: System records attendance data by date, class, and student.
- Data Storage: Store student data including names and phone numbers, for students in the range 463-536
- Summarization tool: Teachers can provide instructions that indicate parameters like date ranges, specific classes or students, etc, and then the tool should summarize attendance data as requested.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust and reliability.
- Background color: Very light lavender (#F0F2FA).
- Accent color: Electric purple (#BF5AF2) for interactive elements and highlights.
- Headline font: 'Space Grotesk' sans-serif font, with a computerized feel. Body font: 'Inter', a modern sans-serif.
- Consistent use of simple, clear icons for navigation and actions.
- Clean, organized layout with clear hierarchy to ensure ease of use for teachers.
- Subtle transitions and feedback animations to enhance user experience.
